# 🟢 Tingkat 1: Pemula (The Foundation)

> ⚠️ **Peringatan:** Ebook ini dihasilkan oleh AI. Mohon laporkan kesalahan yang ditemukan melalui _issues_.

**Fokus:** Memahami filosofi utility-first, instalasi dasar, dan membangun layout sederhana.

Selamat datang di tingkat pemula! Pada tahap ini, kita akan membangun fondasi yang kuat sebelum melangkah ke konsep yang lebih kompleks. Pastikan Anda memahami setiap konsep sebelum melanjutkan ke tingkat berikutnya.

## 📚 Kurikulum Pembelajaran

### 1. Pengenalan & Instalasi
Memahami dasar kerja framework dan cara menyiapkannya di lingkungan pengembangan Anda.
- **Konsep Utility-First:** Memahami mengapa Tailwind berbeda dari CSS tradisional (BEM, OOCSS).
- **Metode Instalasi:**
  - Menggunakan CDN (hanya untuk prototyping/belajar cepat).
  - Instalasi via CLI & NPM (standar industri).
  - Integrasi dengan framework (React, Vue, Laravel, Next.js).
- **Konfigurasi Dasar:** Memahami file `tailwind.config.js` dan `postcss.config.js`.

### 2. Typography & Warna Dasar
Mengelola tampilan teks dan skema warna dalam aplikasi.
- **Teks:** `text-size`, `font-weight`, `text-color`, `text-align`, `leading` (line-height), `tracking` (letter-spacing).
- **Warna:** Palet warna default Tailwind, opacity modifier (contoh: `bg-blue-500/50`).
- **Background:** `bg-color`, `bg-image`, `bg-position`, `bg-size`, gradient.

### 3. Box Model & Spacing
Mengatur jarak dan ukuran elemen untuk menciptakan layout yang rapi.
- **Margin & Padding:** 
  - Menggunakan utility `m-` dan `p-` (contoh: `m-4`, `px-2`, `py-10`).
  - Memahami arah spacing (`t`, `r`, `b`, `l`, `x`, `y`).
- **Width & Height:** Mengatur ukuran elemen (`w-full`, `h-screen`, `max-w-md`).
- **Border & Radius:** Membuat batas elemen dan sudut melengkung (`border`, `rounded-lg`).
- **Shadow:** Menambahkan kedalaman visual (`shadow`, `shadow-xl`).

## 🛠️ Latihan Praktik

Setelah mempelajari materi di atas, kerjakan latihan berikut untuk menguji pemahaman Anda:
1. **Kartu Profil:** Buat kartu profil sederhana dengan foto, nama, dan deskripsi menggunakan utility spacing dan typography.
2. **Tombol Variatif:** Buat 3 jenis tombol (primary, secondary, outline) dengan variasi warna dan padding.
3. **Layout Navbar:** Buat navigasi sederhana dengan logo di kiri dan menu di kanan.
